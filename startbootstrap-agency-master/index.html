Jax Wing Finder

This is a single-page, responsive application built with the React.js framework and hosted on GitHub Pages. I developed a sidebar that links to markers on a Google map showing restaurants serving wings in Jacksonville, FL. The user can view the address and phone number for each location on the map (retrieved by Axios request to FourSquare API).

Other skills used: HTML5 and CSS3

  • Date: October 2018
  • For Udacity Front-End Nanodegree

MyReads Book App

This project was bootstrapped using create-react-app. The React.js framework was used to make this app that categorizes books into shelves. Starter code and API server were provided by Udacity. The book shelf categories are; currently reading, have read, and want to read. The user can also search for more books to add to the categories.

Other skills used: HTML5, CSS3, and Book Lender API

  • Date: September 2018
  • For Udacity Front-End Nanodegree

Restaurant Reviews

A static webpage was given that lacks accessibility for screen readers and responsiveness for different sized displays. A service worker was employed to create an offline experience and accessibility and responsive features were added. Leaflet.js was used with Mapbox for the map on the page.

Other skills used: HTML5, CSS3, JavaScript, and Python server during production

  • Date: September 2018
  • For Udacity Front-End Nanodegree

The Princess Heart Collection

This project is a clone of the classic game Frogger. Udacity provided the art assets and game engine. The player, enemies, and other game entities were coded in JavaScript's object-oriented pseudo-classical style. Enemy sprites were coded to move at different speeds.

Other skills used: JQuery, Bootstrap, HTML5, and CSS3

  • Date: July 2018
  • For Udacity Front-End Nanodegree

Pixel Art Maker

Starter code was provided by Udacity. JavaScript while and for loops were used to clear and create the grid. The makeGrid() function was used and called when user submits the grid size. Event listeners and selectors were accessed to trigger grid creation and modify grid colors. This project was built in the Codepen editor.

Other skills used: JQuery, HTML5, and CSS

  • Date: March 2018
  • For Udacity Front-End Nanodegree
